# 二.常量
使用cosnt 可以声明一个常量，常量一旦赋值就不能修改了
## 1. 常量不能重新赋值
```js
const NAME = 'NAME'
NAME = 'ABC' //Assignment to constant variable
```
## 2. 变量值可以改变
::: warning
const 限制的是不能给变量重新赋值，而变量的值本身是可以改变的
:::
```js
const obj= {a:1,b:3}
obj.b=5
```
## 3. 不同块级作用域可以多次定义
```js
const A = "0";
{
    const A = "A";
    console.log(A)
}
{
    const A = "B";
    console.log(A)
}
console.log(A)
```
::: tip
A B 0
:::
