## 首先安装
```
npm i eslint-plugin-vue -S
```
## .babelrc下配置
```
"plugins": ["transform-vue-jsx", "transform-runtime","vue"]
```
## vscode中`ctr`+`p`搜索setting.json 配置
```
"eslint.autoFixOnSave": true,
"eslint.validate": [
    "javascript",{
        "language": "vue",
        "autoFix": true
    },"html",
    "vue"
]
```

重启项目保存就可以看到自动格式化vue代码
