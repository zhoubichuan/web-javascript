# 单元测试-测试同步代码

## 2.写一个单元测试小案例

### 1.安装

* 全局安装

```
npm install --global mocha
```

* 项目依赖（开发依赖）

```
npm i mocha chai -D
```

### 2.被测试的代码

* calculator.js

```
exports.add = (a, b) => a + b;
exports.minus = (a, b) => a - b;
exports.multiply = (a, b) => a * b;
exports.divide = (a, b) => a / b;
```

### 3.测试用例

* 新建测试文件夹 test
* 新建测试用例 1.test.js

```
const { assert } = require("chai");
const { add, minus, multiply, divide } = require("../calculator");

describe("calulator", () => {
  it("add", () => {
    assert.equal(add(1, 2), 3);
  });
});
```

### 4.运行测试用例

* 项目根目录

```
mocha
```

* 结果

```
calulator
    √ add


  1 passing (13ms)
```
