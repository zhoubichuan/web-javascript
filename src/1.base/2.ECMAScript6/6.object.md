## 2.**proto** 链

```js
console.log("1".__proto__)
```

以前设置原型链

```js
let name = { name: "zbc" }
let age = { age: 9 }
name.__proto__ = age
console.log(name.age)
```

现在设置原型链

```js
Object.setPrototypeOf(name, age)
```

获取原型链

```js
Object.getPrototypeOf(name)
```

可以通过 super 关键字获取父属性

```js
let age = {
  age: 8,
  name: 123,
}
let obj = {
  name: "zbc",
  getName() {
    return super.name
  },
  __proto__: age,
}
console.log(obj.getName())
```
