# 二.数据类型

- 变量声明分为字面量创建和构造函数（实例化）创建
- 构造函数创建 typeof 检测数据类型都为 object
- 原生对象：String，Number，Boolean，Function，object，Array，Date，Error，RegExp，global，Math
- 内置对象：global，Math
- 宿主对象：DOM，BOM
- 全局对象：window（document 属于 window 下）
