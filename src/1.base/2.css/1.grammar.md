## 10.清除浮动

- 为何需要清除浮动：父级出现塌陷,这是由于使用了浮动脱离了文档流，引起了父级塌陷

::: demo

```vue
<style>
.float-eg {
  height: 150px;
}
.float-eg * {
  margin: 0;
  padding: 0;
}
.float-eg li {
  list-style: none;
}
.float-eg ul {
  border: 10px solid red;
  width: 80%;
  box-sizing: border-box;
  margin: 0 auto;
}
.float-eg li {
  width: 25%;
  height: 100px;
  float: left;
}
.float-eg li:nth-child(1) {
  background: gray;
}
.float-eg li:nth-child(2) {
  background: yellow;
}
.float-eg li:nth-child(3) {
  background: black;
}
.float-eg li:nth-child(4) {
  background: green;
}
</style>
<template>
  <div class="float-eg">
    父级：红色区域
    <ul>
      <li></li>
      <li></li>
      <li></li>
      <li></li>
    </ul>
  </div>
</template>
```

:::
