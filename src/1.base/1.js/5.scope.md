# 五.流程控制

## 1.if

用于基于不同的条件来执行不同的动作
`if` 、 `if ... else`、`if ... else if else`

## 2.for

在条件语句为 true 时，可以将代码块执行指定的次数

## 3.`for ... in`

用于遍历数组或者对象的属性（对数组或者对象的属性进行循环操作）

## 4.continue

跳过循环中的一个迭代

```js
for (var i = 0; i < 10; i++) {
  continue
  console.log(i)
}
console.log(i)
```

## 4.break

用于跳出循环

```js
for (var i = 0; i < 10; i++) {
  console.log(i)
  break
}
console.log(i)
```

## 5.switch

用于基于不同的条件来执行不同的动作

## 6.while

当条件语句为 true 时，执行语句块

## 7.do ... while

执行一个语句块，在条件语句为 true 时继续执行该语句块
