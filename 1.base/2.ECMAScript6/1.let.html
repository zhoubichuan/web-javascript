# 一.let

前言

  • var 定义的变量没有块的概念，可以跨块访问，不能跨函数访问，有变量提升，可重复声明
  • let 定义的变量，只能在块作用域里访问，不能跨块访问，也不能跨函数访问，无变量提升，不可以重复声明
  • let 声明的变量只在块级作用域内有效，不存在变量提升，不可以重复声明
  • 或者说 let 变量提升了，但是在 let 声明变量前不能使用该变量，这特性叫暂时性死区
  • 如果有重复变量let会在编译阶段报错

# 1.1 重复声明

}

# 2.作用域

# 2.1 块级作用域

  • 允许块级作用域任意嵌套
  • 外层作用域无法读取内层作用域的变量
  • 内层作用域可以定义外层作用域的同名变量
  • 函数本身的作用域在其所在的块级作用域之内
"use strict"
function fn() {
  console.log("out")
}
;(function () {
  if (false) {
    function fn() {
      console.log("in")
    }
  }
  fn()
})()
